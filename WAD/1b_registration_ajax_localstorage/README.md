# WAD Assignment 1b - User Registration With AJAX and LocalStorage

## Aim
Write a JavaScript program to get user registration data, push it into an array/localStorage, send it using AJAX POST method, and display the data list on a new page.

## Files
- `index.html` - Registration form
- `list.html` - Registered users list page
- `script.js` - JavaScript logic for array, localStorage, AJAX POST and display
- `style.css` - Page styling

## Run
Open `index.html` in a browser.

## Features
- Accepts user registration data
- Stores data in a JavaScript array
- Saves data in browser localStorage
- Sends data using AJAX POST
- Displays registered users on `list.html`

## Technologies Used
- HTML
- CSS
- JavaScript
- AJAX
- LocalStorage

## Conclusion
User registration data was collected, stored locally, sent using AJAX POST, and displayed on a separate page.
