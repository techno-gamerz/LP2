# WAD Assignment 1a - Ecommerce Admin Dashboard

## Aim
Create a responsive ecommerce admin dashboard with sidebar and statistics cards using HTML, CSS and Bootstrap.

## Files
- `index.html` - Dashboard structure using Bootstrap
- `style.css` - Custom styling and responsive sidebar CSS

## Run
Open `index.html` in a browser.

## Features
- Responsive sidebar
- Ecommerce dashboard header
- Statistics cards
- Recent orders table
- Product and order status sections

## Technologies Used
- HTML
- CSS
- Bootstrap 5

## Conclusion
A responsive ecommerce dashboard was created using HTML, CSS and Bootstrap.
