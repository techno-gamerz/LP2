# WAD Assignment 2a - GitHub Version Control

## Aim
Create a version control account on GitHub, create a repository using Git commands, and push code to GitHub.

## Prerequisites
- Git installed
- GitHub account created
- Repository created on GitHub

## Commands

```bash
git --version
git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"

git init
git status
git add .
git commit -m "Initial commit"

git branch -M main
git remote add origin https://github.com/username/repository-name.git
git push -u origin main
```

## Useful Commands

```bash
git status
git log --oneline
git remote -v
git add filename
git commit -m "message"
git push
git pull
```

## Conclusion
The project code was added to a local Git repository and pushed to a remote GitHub repository using Git commands.
