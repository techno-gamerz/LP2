# WAD Mini Project - Ecommerce Web Application

## Domain
Ecommerce

## Aim
Develop a full stack ecommerce web application using frontend and backend technologies.

## Features
- View products
- Add product
- Place order
- Store product and order data in MongoDB

## Backend Run

```bash
cd backend
npm install
npm start
```

Backend URL:

```text
http://localhost:5000
```

## Frontend Run
Open `frontend/index.html` in a browser.

## APIs

| Method | URL | Description |
|---|---|---|
| GET | /api/products | Get products |
| POST | /api/products | Add product |
| POST | /api/orders | Place order |
| GET | /api/orders | Get orders |

## MongoDB

```text
mongodb://127.0.0.1:27017/wad_mini_project
```
