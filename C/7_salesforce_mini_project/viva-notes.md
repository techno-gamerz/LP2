# Viva Notes

## What is Salesforce?
Salesforce is a cloud-based CRM and PaaS platform used to build business applications.

## What is a Custom Object?
A custom object stores application-specific data in Salesforce.

## What is a Field?
A field stores one attribute of an object, such as email, mobile number or status.

## What is a Lookup Relationship?
A lookup relationship connects one object record to another object record.

## What is a Lightning App?
A Lightning App is a collection of tabs, objects and pages for a business process.

## What is a Report?
A report displays filtered and grouped records from Salesforce objects.

## What is a Dashboard?
A dashboard visually represents report data using charts and metrics.
